Implementation and Experiments
• ‘LinearRegression’ has one variable
– w: weight vector learned by linear regression
• ‘KNN’ has one variable
– k: the k value chosen for k-NN
• The methods commonly used by both ‘LinearRegression’ and ‘KNN’
are defined under the parent class ‘ML’
– setData:
○ Reads data file and creates data matrices (createMatrices)
○ Sets the class variables with appropriate data matrices
– createMatrices:
○ Reads data file, creates data matrices using ndarray of NumPy,
and returns them

Implementation and Experiments
– testModel:
○ Predicts the target value of each of the test examples
(runModel) and stores them in testPy
– report:
○ Calculates the root mean-squared error by comparing the
predicted values (testPy) with the real targets (testDy) and
displays it
• The ‘LinearRegression’ subclass has two methods
– buildModel:
○ Builds a linear regression model using the training data and
stores the learned weight vector
Linear Regression & k-NN (Part B)

Implementation and Experiments
– runModel:
○ Applies the linear regression model to a test data by taking an
inner product of the leaned weight vector and the test data
• The ‘KNN’ subclass has six methods
– buildModel:
○ Simply stores the value of k
– runModel:
○ Applies k-NN to a query example by identifying k examples
closest to the query (findCK) and taking the average of their
target values (takeAvg)
Linear Regression & k-NN (Part B)
8
Implementation and Experiments
– findCK:
○ Idendifies k examples closest to the given query example
ü Prepare a k ´ 2 matrix and fill it with the first k examples in the
training set
» Example index stored in the first column
» Squared distance to the query stored in the second column
(sDistance)
ü While scanning through the training set, update the matrix whenever
an example is found whose distance to the query is shorter than any
of the examples in the matrix (updateCK)
○ Returns the found k closest examples
Linear Regression & k-NN (Part B)
9
Implementation and Experiments
– sDistance:
○ Calculates and returns the squared distance between two given
examples
– updateCK:
○ If the example with index i is found closer to the query than any
of the examples in the set of k closest examples, the latter is
replaced by the former
– takeAvg:
○ Calculates and returns the average of the target values of the k
closest examples
